# Interactive-Dashboard-
Developed an interactive Power BI Sales Dashboard to analyze ABC Company’s sales performance (2020–2022). The dashboard provides insights into sales, profit, cost, quantity, transactions, product categories, brands, and regional performance using DAX, Power Query, KPI cards, maps, and interactive filters.
